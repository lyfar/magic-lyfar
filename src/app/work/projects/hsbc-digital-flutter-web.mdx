## Project Overview

A fast, end-to-end prototype of an HSBC-style digital assets experience built with Flutter. This project was developed as part of a job application process to demonstrate Flutter capabilities and rapid prototyping skills. It explores adaptive Material/Cupertino design, mock authentication, market browsing, asset details with a chart, a buy flow with review/confirmation, a portfolio view with recent activity, and persistent theme settings.
